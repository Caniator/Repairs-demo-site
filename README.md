# Aerones — Landing Page

A marketing landing page for [Aerones](https://aerones.com), the robotic wind turbine care systems company.

## Overview

Single-file HTML landing page featuring:
- Responsive navigation with smooth scroll
- Hero section with CTA
- Services overview (Inspection, Cleaning, Repair, Coating, Offshore)
- Stats / social proof section
- Scroll-triggered fade-in animations
- Contact / footer with US and EU office info

## Stack

- Pure HTML + CSS + vanilla JS (no build step, no dependencies)
- Google Fonts: Roboto, Roboto Condensed, Roboto Mono
- CSS custom properties for theming
- IntersectionObserver for scroll animations

## Usage

Just open `index.html` in a browser — no server required.

```bash
open index.html
# or
python3 -m http.server 8080
```

## Deployment

Works with any static host: GitHub Pages, Netlify, Vercel, Cloudflare Pages, etc.

### GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Your site will be live at `https://<username>.github.io/<repo>/`
